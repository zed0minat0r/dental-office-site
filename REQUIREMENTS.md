# Bright Smile Dental — Requirements

## Core Sections (Priority Order)

### P0 — Must Have
1. Hero with appointment booking CTA
2. Services list (general, cosmetic, emergency, pediatric, etc.)
3. About the practice / Meet the team
4. Patient testimonials/reviews
5. Location & hours with map placeholder
6. Contact form / appointment request
7. Insurance accepted section

### P1 — Should Have
1. Before/after gallery (CSS placeholders)
2. New patient information section
3. FAQ section
4. Special offers / new patient discount
5. Blog/dental tips section

### P2 — Nice to Have
1. Virtual office tour placeholder
2. Patient portal link
3. Emergency dental care section with phone number
4. Smile gallery

## Technical Requirements
- Single-page HTML/CSS/JS (no frameworks)
- Fast load times
- WCAG accessible
- Mobile-first responsive
- Dark mode support
- Semantic HTML5
